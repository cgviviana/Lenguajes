Curso HTML

1. ¿Qué es HTML?
Es el lenguaje que se emplea para el desarrollo de páginas de internet, el cual esta constituido de elementos que el navegador interpreta y las despliega en la pantalla de acuerdo a su objetivo. .

2. Estructura interna de una página HTML

<head >
   <title > Título de la página </title >
   <meta charset="UTF-8"
</head >
<body >
Cuerpo de la página </body >
</html >

3. Salto de línea (<br>)
<br> Todo el texto que disponemos en el cuerpo de la página aparece en la misma línea, un navegador no tiene en cuenta la tecla ENTER por lo que esta etiqueta, es para indicarle que queremos que continúe en la próxima línea .

Ejemplo: Estimado amigo, <br> Solo quería saludarte.
Estimado amigo,
Solo quería saludarte.
4. Párrafo (<p>, </p>)
<p> y <p> Un párrafo es una oración o conjunto de oraciones referentes a un mismo tema. Todo lo que encerremos en estas etiquetas aparecerá separado por un espacio con respecto al próximo párrafo..

Ejemplo: <p>La importancia de reconocer cada una de las etiquetas de html, permite que podamos estructurar páginas organizadas y con un adecuado estilo.</p> <p> Pese a lo dicho anteriormente, se debe tener en cuenta que html por sí solo, no genera una página dinámica tiene que, por lo menos combinarse el lenguaje CSS y Java Script para que sea funcional.</p>

La importancia de reconocer cada una de las etiquetas de html, permite que podamos estructurar páginas organizadas y con un adecuado estilo.

Pese a lo dicho anteriormente, se debe tener en cuenta que html por sí solo, no genera una página dinámica tiene que, por lo menos combinarse el lenguaje CSS y Java Script para que sea funcional.

7. Enlace interno (<a></a>)
Se usa para enlazar a otra página del mismo sitio web.

Ejemplo: <a href="pagina.html">Ir a otra página</a>
Ir a otra página
8. Enlace externo (<a></a>)
Permite enlazar a sitios web de fuera del sitio.

Ejemplo: <a href="https://google.com">Google</a>
Google
9. Imagen (<img>)
Muestra una imagen usando el atributo src y texto alternativo con alt.

Ejemplo: <img src="1.jpg" alt="Descripción">
10. Imagen con enlace (<a><img></a>)
Puedes hacer clic sobre la imagen para ir a un enlace.

Ejemplo: <a href="https://google.com"><img src="1.jpg"></a>
11. Nueva pestaña (<a target="_blank">)
Para abrir el enlace en otra pestaña del navegador.

Ejemplo: <a href="https://google.com" target="_blank">Ir a Google</a>

23. Contenido de la cabecera de la página (<title>, </title>)
Es una etiqueta que sirve para ponerle nombre a la página, y ese nombre aparece en la parte de arriba del navegador, donde están las pestañas, no dentro del cuerpo de la página

Ejemplo:

<!DOCTYPE html>
  <html>
    <head>
      <title> Juego Trivia </title>
      <meta charset="UTF-8">
    </head>
    <body>

    </body>
  </html>
24. Contenido de la cabecera de la página (<meta >)
Se usa para dar información sobre tu página, pero no se muestra en la pantalla, solo los navegadores y los buscadores pueden ver.
  • <meta charset="UTF-8">Le dice al navegador qué letras y símbolos puede usar la página.
  • <meta name="keywords" content="Temas" > Palabras clave para que los buscadores sepan tema de la página.
  • <meta name="description" content="Descripción">Es una mini-descripción que aparece en el buscador cuando alguien busca la página.
  • <meta name="author" content="Autor de la página"> Dice quién hizo la página.
  • <meta name="copyright" content="Dueño de la página"> Dice quién es el dueño de los derechos de la página.
Ejemplo:

    <head>
      <title> Juego Trivia </title>
      <meta charset="UTF-8">
      <meta name="keywords" content="html, programación, Tutorial">
      <meta name="description" content="Aprendiendo html">
      <meta name="author" content="La Profe Vivy">
      <meta name="copyright" content="Viviana R. T.">
    </head>
25. Comentarios dentro de una página <!-- -->
Comentario es un texto que se escribe en el código pero no es código, es como un mensaje recordatorio o explicativo que no interfiere la programación ni se muestra en la página web.

Ejemplo

<!-- Este es un comentario de una línea-->
<!-- Este es un comentario de
        varias
        líneas-->
26. Sintaxis para caracteres especiales
Hay algunos símbolos como <, > o & que HTML usa para su propio lenguaje, por eso no se pueden escribir directamente en una página web o el navegador se confunde, por lo que se usa códigos especiales llamados entidades HTML, cuya estructura es: &(nombre); .

Caracter Nombre Entidad
< Menor que &lt;
> Mayor que &gt;
& Ampersand &amp;
© Derechos de autor &copy;
espacio en blanco &nbsp
" Comillas dobles &quot;
' Apóstrofo &apos;


Ejemplo

<!DOCTYPE html>
<html>
   <head>
        <title>Título de la página</title>
        <meta charset="UTF-8">
    </head>
    <body>
       <h1> 10+x*y < 12*z </h1>
    </body>
</html>
Título de la página

10+x*y < 12*z

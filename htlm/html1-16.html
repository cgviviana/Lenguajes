Curso HTML

1. ¿Qué es HTML?
Es el lenguaje que se emplea para el desarrollo de páginas de internet, el cual esta constituido de elementos que el navegador interpreta y las despliega en la pantalla de acuerdo a su objetivo. .

2. Estructura interna de una página HTML
<!DOCTYPE html>
<html >
<head >
   <title > Título de la página </title >
   <meta charset="UTF-8"
</head >
<body >
Cuerpo de la página </body >
</html >


Ejemplo:
Cuerpo de la página
3. Salto de línea (<br>)
<br> Todo el texto que disponemos en el cuerpo de la página aparece en la misma línea, un navegador no tiene en cuenta la tecla ENTER por lo que esta etiqueta, es para indicarle que queremos que continúe en la próxima línea .

Ejemplo: Estimado amigo, <br> Solo quería saludarte.
Estimado amigo,
Solo quería saludarte.
4. Párrafo (<p>, </p>)
<p> y <p> Un párrafo es una oración o conjunto de oraciones referentes a un mismo tema. Todo lo que encerremos en estas etiquetas aparecerá separado por un espacio con respecto al próximo párrafo..

Ejemplo: <p>La importancia de reconocer cada una de las etiquetas de html, permite que podamos estructurar páginas organizadas y con un adecuado estilo.</p> <p> Pese a lo dicho anteriormente, se debe tener en cuenta que html por sí solo, no genera una página dinámica tiene que, por lo menos combinarse el lenguaje CSS y Java Script para que sea funcional.</p>

La importancia de reconocer cada una de las etiquetas de html, permite que podamos estructurar páginas organizadas y con un adecuado estilo.

Pese a lo dicho anteriormente, se debe tener en cuenta que html por sí solo, no genera una página dinámica tiene que, por lo menos combinarse el lenguaje CSS y Java Script para que sea funcional.

Pone el encabezamiento del texto en diferentes tamaños o posiciones según jerarquía.

Ejemplo: <h1>Título 1 </h1> <h2>Título 2 </h2> <h3>Título 3 </h3> <h4>Título 4 </h4> <h5>Título 5 </h5> <h6>Título 6 </h6>

6. Énfasis (<strong>, <em>)
<strong> pone el texto en negrita. <em> lo pone en cursiva para destacar.

Ejemplo: <strong>Importante</strong> <em>Destacado</em>
Importante y Destacado
7. Enlace interno (<a></a>)
Se usa para enlazar a otra página del mismo sitio web.

Ejemplo: <a href="pagina.html">Ir a otra página</a>
Ir a otra página
8. Enlace externo (<a></a>)
Permite enlazar a sitios web de fuera del sitio.

Ejemplo: <a href="https://google.com">Google</a>
Google
9. Imagen (<img>)
Muestra una imagen usando el atributo src y texto alternativo con alt.

Ejemplo: <img src="1.jpg" alt="Descripción">
10. Imagen con enlace (<a><img></a>)
Puedes hacer clic sobre la imagen para ir a un enlace.

Ejemplo: <a href="https://google.com"><img src="1.jpg"></a>
11. Nueva pestaña (<a target="_blank">)
Para abrir el enlace en otra pestaña del navegador.

Ejemplo: <a href="https://google.com" target="_blank">Ir a Google</a>
Ir a Google
12. Correo (<a href="mailto:">)
Crea un botón para enviar un correo electrónico.

Ejemplo: <a href="mailto:correo@ejemplo.com">Enviar correo</a>
Enviar correo
13. Ancla interna (<a href="#seccion">)
Te permite ir a una parte específica de la misma página.

Ejemplo: <a href="#final">Ir al final</a>
Ir al final
14. Ancla externa (<a href="pagina.html#parte">)
Lleva a una parte específica en otra página.

Ejemplo: <a href="pagina.html#seccion">Ver sección</a>
Ver sección
15. Lista ordenada (<ol></ol>)
Crea una lista numerada.

Ejemplo: <ol><li>Uno</li><li>Dos</li></ol>
  1. Uno
  2. Dos
16. Lista no ordenada (<ul></ul>)
Crea una lista con viñetas.

Ejemplo: <ul><li>Pan</li><li>Leche</li></ul>
  • Pan
  • Leche
